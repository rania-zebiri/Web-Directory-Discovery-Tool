# Web Directory Discovery Tool

## 📝 Description
A Python tool used to discover hidden directories and files on a web server. This is a fundamental step in the "Reconnaissance" phase of a Web CTF challenge.

## ✨ Features
- **Status Code Detection:** Identifies 200 (OK) and 403 (Forbidden) responses.
- **Custom Wordlists:** Supports any `.txt` wordlist provided by the user.
- **Timeout Handling:** Prevents the script from hanging on slow servers.
